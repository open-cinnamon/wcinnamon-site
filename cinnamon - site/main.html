Wide Cinnamon

A new way to work in teams

Wide Cinnamon is a virtual environment.
It uses your system to create a subsystem that is isolated from your root system.
Most applications are coded in GDScript for better compatibility across systems, but other methods can also be explored.
Wide Cinnamon is based on Godot 4.3 with C# support.
However, this does not mean you can only use C# for your own projects. A significant part of Wide Cinnamon is written in C#, while some built-in tools are developed in GDScript.
The main goal of Wide Cinnamon is to provide seamless connectivity between different systems.

Here are some of the built-in applications:

  • error: A simple application which takes arguments and prints them. Arguments are logged to the system log file.
  • cig: Another simple application that can manage Cig Modules. More about Cig Modules
