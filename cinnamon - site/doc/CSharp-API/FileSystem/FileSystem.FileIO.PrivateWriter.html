WC#-API/FileSystem

Wcinnamon.Syslib.FileSystem.FileIO.Writer

The Writer class provides functionality to write data to a file using C#'s System.IO.TextWriter as a base. While Godot's FileAccess class already provides robust file I/O capabilities, Writer can be particularly useful when you need enhanced interaction with standard C# libraries.

Namespace

Wcinnamon.Syslib.FileSystem.FileIO

Inheritance

System.IO.TextWriter

Properties

  • Encoding (override): Returns the encoding used for writing. Defaults to UTF-8.

Methods

  • Open(string path)

    Opens the specified file for writing.

    • Parameters: string path - The path to the file to be opened.
    • Exceptions: Throws an exception if the file cannot be opened.
  • Write(char value) (override)

    Writes a single character to the file.

    • Parameters: char value - The character to write.
    • Exceptions: Throws System.NullReferenceException if the file is not opened.
  • Write(string value) (override)

    Writes a string to the file.

    • Parameters: string value - The string to write.
    • Exceptions: Throws System.NullReferenceException if the file is not opened.
  • Write(char[] buffer, int index, int count) (override)

    Writes a range of characters from a buffer to the file.

    • Parameters:
      • char[] buffer - The character buffer containing data to write.
      • int index - The starting position in the buffer.
      • int count - The number of characters to write.
    • Exceptions: Throws System.NullReferenceException if the file is not opened.
  • Close() (override)

    Closes the currently opened file. Ensures the file is properly released.

  • Dispose(bool disposing) (protected override)

    Disposes of the resources used by the Writer object.

Usage Example

// Create an instance of Writer
Writer writer = new Writer();

// Open a file for writing
writer.Open("example.txt");

// Write data to the file
writer.Write("Hello, World!");

// Close the file
writer.Close();

Remarks

Ensure that the Open method is called before attempting to write data to a file. Calling Write methods without opening a file will result in a System.NullReferenceException.
